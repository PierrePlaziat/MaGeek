# MaGeek

WPF MTG Library Manager 

## Description

MTG card collection manager including
- Desktop Client
- Desktop Server
- Deployable Server
Powered by
- MtgJson
- Scryfall
Stack
- Wpf
- AvalonDock
- Identity Framework
- Entity Framework 
- sqlite
- Docker

## Functionalities

-	Exaustive MTG informations
-	Collection gestion
-	Deck building
-	Printing
-	Estimations
