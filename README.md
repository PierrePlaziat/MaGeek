# MaGeek

WPF MTG Library Manager 

## Description

I needed a tool to manage my MTG card collection on local drive, that reflect my needs, which can eventually evolve.

## Functionalities

-	Exaustive cards infos
-	Advanced Search
-	Collection gestion
-	Proxy printing
-	Price estimations
-	Customizable UI
-	Import/Export 
-	preco and set exploration
-	statistics
